# Kajal's MealMate AI 🍽️

An AI-powered Indian meal recommendation system built using Python, Streamlit, Pandas, and Scikit-Learn.

Features:
- Breakfast, Brunch, Lunch, Snacks, Dinner recommendations
- Cuisine-based filtering
- Vegetarian & Non-Vegetarian support
- Goal-based recommendations
- Mood-based recommendations
- Premium UI

Tech Stack:
- Python
- Streamlit
- Pandas
- Scikit-Learn
